Smart Form Guide - Chrome Extension Icons

How to Generate PNG Icons

  1. Right-click on each icon below and select "Save image as..."
  2. Save them with the following names in the chrome-extension/icons/ folder:
    • 16x16 icon → icon16.png
    • 32x32 icon → icon32.png
    • 48x48 icon → icon48.png
    • 128x128 icon → icon128.png
  3. Update the manifest.json to include the icons section
  4. Delete this HTML file after generating the icons

Alternative: Use Online Converter

You can also:

  1. Copy the SVG code from chrome-extension/icons/icon.svg
  2. Use an online SVG to PNG converter (like convertio.co or cloudconvert.com)
  3. Generate PNG files in the required sizes: 16x16, 32x32, 48x48, and 128x128 pixels
  4. Save them as icon16.png, icon32.png, icon48.png, and icon128.png
